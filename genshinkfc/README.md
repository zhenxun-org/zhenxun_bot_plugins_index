# 原神KFC语音插件（呐呐呐，服务员欧尼酱~~......异世相遇，尽享美味！）
（异世相遇，尽享社死！）
需要安装的依赖
```
pip install ffmpeg
```

    在QQ群发送指定的指令，Bot会随机选择一条语音回复
    指令：
        KFC/kfc/原神KFC/原神kfc/二次元KFC/二次元kfc/异世相遇尽享美味
